# Special Guest - Band Website

Official website for the band **Special Guest** (winner of the SRF ESSC 2026 Swiss school song contest).

### 🔗 Live Website
You can visit the live website here:
👉 **[specialguest-maria.github.io/Special-Guest](https://specialguest-maria.github.io/Special-Guest/)**

---

## Technical Stack
- **Frontend:** HTML5, Vanilla CSS3, Javascript (ES6)
- **Data Engine:** Loaded dynamically from `/docs/content_data.json`
- **Hosting:** GitHub Pages (serving from the `/docs` folder)

## Local Development
Since this is a fully static website, you can run it locally by launching any local server (e.g., Python HTTP server, VS Code Live Server) or simply opening the [docs/index.html](docs/index.html) file in your browser.
